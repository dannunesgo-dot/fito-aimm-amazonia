# AIMM ↔ WB Gaps Register

## Objetivo
Registrar lacunas entre indicadores AIMM e disponibilidade/aderência de indicadores da World Bank.

## Escala de severidade
- **Alta**: sem indicador WB utilizável (direto/proxy insuficiente)
- **Média**: há proxy, mas com risco metodológico relevante
- **Baixa**: há indicador utilizável com ajustes menores

| GAP-001 | CATEGORIA_01 |  |  | Sem equivalente direto | Alta |  |  |  |  |  |  | Aberto |

## Tipos de Lacuna (padrão)
- Sem equivalente direto
- Unidade incompatível
- Frequência inadequada
- Cobertura geográfica insuficiente
- Série temporal insuficiente
- Definição metodológica divergente

## Bases complementares possíveis
- Microdados setoriais nacionais
- Bases regulatórias do setor
- Pesquisas amostrais oficiais
- Dados operacionais internos IFC/projeto (quando permitidos)

## Critério de fechamento de lacuna
Uma lacuna só pode ser fechada quando:
1. houver indicador validado (ou proxy aprovado),
2. justificativa técnica estiver registrada,
3. impacto no score/calculadora estiver documentado.

2. Como os usuários devem trabalhar

  1. Reunir artigos, notas técnicas, planilhas, PDFs, DOCX e arquivos GIS na pasta de entrada do Google Drive.
  2. Copiar o link ou ID da pasta de entrada.
  3. Preencher os campos abaixo para preparar a execução.
  4. Enviar os dados ao operador responsável.
  5. O operador executa a automação pelo GitHub Actions.
  6. O sistema gera relatório técnico, relatório executivo, matrizes e evidências no Drive.
Esta interface não guarda dados e não recebe upload direto. Ela prepara a execução sem expor credenciais. O processamento real ocorre no workflow protegido do GitHub Actions.
